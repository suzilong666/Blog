引入方式与优先级 ​

引入方式 ​

html
 /* 1. 行内样式 */
<div style="color: red;"></div>

/* 2. 内部样式表 */
<style> p { color: blue; } </style>

/* 3. 外部样式表（推荐） */
<link rel="stylesheet" href="style.css">

/* 4. @import方式（不推荐） */
@import url("other.css");

优先级 ​

行内样式 > 内部样式表 > 外部样式表 > import
