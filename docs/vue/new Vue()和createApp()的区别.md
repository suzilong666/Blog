# new Vue()和createApp()的区别

| 维度         | new Vue()                   | createApp()             |
| ------------ | --------------------------- | ----------------------- |
| 全局污染     | 存在（共享全局 Vue 实例）   | 无污染（每个 app 独立） |
| 插件机制     | 污染所有实例                | 仅影响当前 app          |
| 多实例       | 难以并存（配置冲突）        | 完美隔离                |
| Tree-shaking | 不支持（全局 API 无法移除） | 支持（按需引入）        |
| 根节点       | 自动挂载（$mount 或 el）    | 显式挂载（.mount()）    |
