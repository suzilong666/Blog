# 引入方式与优先级

## 引入方式
```html
 /* 1. 行内样式 */
<div style="color: red;"></div>

/* 2. 内部样式表 */
<style> p { color: blue; } </style>

/* 3. 外部样式表（推荐） */
<link rel="stylesheet" href="style.css">

/* 4. @import方式（不推荐） */
@import url("other.css");
```

## 优先级

行内样式 > 内部样式表 > 外部样式表 > import

## link和import的区别

| 特性        | link标签                 | @import                     |
| ----------- | ------------------------ | --------------------------- |
| 用法        | 在HTML文档的head部分使用 | 在CSS文件或style标签内使用  |
| 加载顺序    | 页面加载时立即加载样式表 | 在加载包含它的css文件后加载 |
| 浏览器支持  | 支持所有主流浏览器       | 支持IE5和所有现代浏览器     |
| 性能        | 加载并行运行，速度快     | 加载顺序依赖，速度慢        |
| dom可操作性 | 可通过js操作和控制       | 不易通过js操作              |
| 样式权重    | 样式权重相同             | 样式权重相同                |
