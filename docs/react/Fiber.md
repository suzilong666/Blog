# Fiber

## Fiber 是什么？

React Fiber 是 React 16 中引入的全新协调引擎（reconciliation engine），它的核心设计目标是**实现可中断的渲染**，使 React 能够更好地利用浏览器的主线程，从而提升复杂应用的性能和用户体验。

## 为什么需要 Fiber？

在 React 16 之前，协调过程是**同步递归**的。一旦开始更新，就会一气呵成地遍历整个组件树，期间无法中断。如果组件树很大或更新频繁，就会长时间占用主线程，导致页面掉帧、动画卡顿、用户输入得不到及时响应。

## Fiber 的核心思想

Fiber 将原本不可分割的更新任务拆分成一个个小的工作单元（unit of work），并为每个单元赋予优先级。浏览器可以按需暂停、恢复或中止这些工作单元，优先响应用户交互等高优先级任务。
