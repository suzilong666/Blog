# 数据类型

JavaScript 是动态类型语言，数据类型主要分为两大类：原始类型（Primitive Types） 和 引用类型（Reference Types/Object Types）。

## 一、原始类型（Primitive Types，7种）

原始类型的值是不可变的，直接存储在**栈内存**中。

### 1. Number - 数字类型

JavaScript只有一种数值类型：number（数字），包括“整数”和带小数的十进制数。此处“整数”之所以加引号是因为和其他语言不同，JavaScript没有真正意义上的整数，这也是它一直以来唯一诟病的地方。

JavaScript中的数字类型是基于IEEE754标准来实现的，该标准通常也被称为“浮点数”。JavaScript使用的是*“双精度浮点数”格式*（64位二进制）。

```js
let age = 25;          // 整数
let price = 99.99;     // 浮点数
let negative = -10;    // 负数
let infinite = Infinity; // 无穷大
let notNumber = NaN;   // 非数字（特殊值）
```

### 2. String - 字符串类型

```js
let name = "Alice";
let greeting = 'Hello';
let template = `Hello, ${name}!`; // 模板字符串
let char = "A"; // JavaScript 没有单独的字符类型
```

### 3. Boolean - 布尔类型

```js
let isLogged = true;
let hasPermission = false;
```

### 4. Undefined - 未定义

```js
let x;                 // 声明但未赋值，值为 undefined
let y = undefined;     // 显式赋值为 undefined
```

### 5. Null - 空值

```js
let emptyValue = null; // 表示"无值"或"空对象指针"
```

### 6. Symbol - 符号类型（ES6新增）

唯一且不可变的值，常用于对象属性的键。

```js
let sym1 = Symbol("id");
let sym2 = Symbol("id");
console.log(sym1 === sym2); // false，每次创建都是唯一的

// 用作对象属性键
let user = {
  [Symbol("id")]: 123
};
```

### 7. BigInt - 大整数类型（ES2020新增）

表示任意精度的整数。

```js
let bigNumber = 1234567890123456789012345678901234567890n;
let anotherBigInt = BigInt("9007199254740991"); // 超过 Number 的安全范围
```

## 二、引用类型（Object Types）

引用类型的值是可变的，存储在**堆内存**中，变量保存的是**内存地址**。

### 1. Object - 对象

```js
// 对象字面量
let person = {
  name: "John",
  age: 30,
  isStudent: false
};

// 构造函数
let car = new Object();
car.brand = "Toyota";
```

### 2. Array - 数组

```js
let fruits = ["apple", "banana", "orange"];
let mixed = [1, "hello", true, {x: 10}];
```

### 3. Function - 函数

```js
// 函数声明
function add(a, b) {
  return a + b;
}

// 函数表达式
let multiply = function(x, y) {
  return x * y;
};

// 箭头函数（ES6）
let divide = (a, b) => a / b;
```

### 4. Date - 日期

```js
let now = new Date();
let specificDate = new Date("2024-01-01");
```

### 5. RegExp - 正则表达式

```js
let pattern = /hello/gi;
let regexp = new RegExp("world", "i");
```

### 6. 其他内置对象

- Map、Set（ES6）
- Promise（ES6）
- Error 及其子类
