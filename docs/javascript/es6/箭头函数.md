# 箭头函数

## 箭头函数是什么？

箭头函数是 ES6 引入的一个新特性，它是一个**纯粹的函数**。

**语法：**

```js
let add = (a,b) => a + b

// 等同于
function add(a, b) {
    return a+ b
}
```

## 箭头函数的特性

### 没有自己的this对象

### 不可以当作构造函数

### 没有 prototype 属性

### 不可以使用arguments对象

### 不可以使用yield命令

### 不能提升

箭头函数是函数表达式，不会提升，必须在定义后调用
